Flash XML Touch Slider

width and height values of the SWF file ( flashmo_237_touch_slider.swf ) can be adjusted from embedded HTML tags. All photos must have the same dimensions of width and height values.

Default XML file name

flashmo_237_photo_list.xml ( or any file name can be set in SWF embedded HTML tags using FlashVars )

Optional Variables in XML <config> tag

auto_play = true by default ( or set it as false )

auto_play_duration = 3 seconds by default ( or any number of seconds )

previous_button = true by default ( or set false to hide it )

next_button = true by default ( or set false to hide it )

slide_direction = left by default ( or right or up or down )

tween_duration = 0.5 seconds by default ( or any number of seconds )

tween_effect = easeInOutQuad by default | easeInOutSine | easeInOutExpo ( or see below Tweener transition cheat sheet )
